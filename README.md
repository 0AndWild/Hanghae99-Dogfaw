### 개발바닥은?

- 개발자 또는 개발자를 꿈꾸는 이들을 위한 협업 서비스입니다.

- Stack별 카테고리를 선택하고 해당 Stack에 익숙한 개발자들을 모아 함께 협업을 할 수 있습니다.
